Based on the files traversing the repository, here is the entire tech stack for the ZahnPrep platform: 
Frontend 
The frontend is a modern web application built for signup, login, dashboard uploads, and revision flows. 
Core Framework: Next.is (v16.2.1) using React (v19.2.4) Language: TypeScript Styling: ailwind CSS (v4) State Management: Zustand (for global state management like exam flows) Backend/Database SDK: @supabase/supabase-js 
Backend 
The backend is a high-performance Python application handling authenticated MCQ generation, processing files, and credit management. 
Core Framework: FastAPI (0.135.2) Language: Python ASGI Server: Uvicorn (0.42.0) Al Integration: google-genai (For the Al pipeline / MCQ generation via Gemini) Payments: razorpay (For Razorpay payment gateway integration) Data Validation: pydantic & pydantic -settings File Uploads: python-multipart 
Database & Authentication (BaaS) 
Platform: Supabase The application relies heavily on Supabase for PostgreSQL database, authentication, and likely storage (as 
hinted by the supabase_schema .sql presence). 
DevOps & CI/CD 
CI/CD: GitHub Actions (.github/workflows/ci.yml) is used for baseline continuous integration (linting, build verification, and smoke testing).
